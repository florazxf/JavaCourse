# Description
这是一个简单的记账软件，主要的功能就是记录收入和支出，可以查看收支明细
界面如下

-------------收支记账软件-------------  
             1 收支明细  
             2 登记收入  
             3 登记支出  
             4 退   出  
             请选择(1-4):
